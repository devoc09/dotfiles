devoc's dotfiles repository
===========================

### Clone dotfiles from devoc09's repository

```
$ git clone https://github.com/devoc09/dotfiles.git
```

### zsh setup

```
$ make zshset
```

### tmux setup

```
$ make tmuxset
```

### neovim setup

```
$ make nvimset
```

### alacrittyset(you need to install Alacritty beforehand)

```
$ make alacrittyset
```
